# Sales-Optimization
Walmart Store Sales Forecasting to Minimize Loss

This study investigates optimal inventory decisions under stochastic demand. Method: A convex
optimization model based on the Newsvendor framework is formulated using Walmart sales data. Results: The
optimal order quantity is derived via the critical fractile, showing sensitivity to cost asymmetry and demand
variability. Conclusion: Stochastic modeling significantly improves inventory efficiency, though limitations arise
from distributional assumptions.
